#### How to use
```js
// 흐름의 속도를 변경하는 방법
this.stop();
this.setGroupPropertyValue("tweenInfo","duration",1000);
this.play();

// 흐름 자체의 사이즈를 변경하는 방법
this.stop();
this.setGroupPropertyValue("tweenInfo","streamSize", 100)
this.play();
```
